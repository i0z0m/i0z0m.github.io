make GitHub Pages by Sphinx¶

Date

2021/02/02 19:20

usage¶

$ pip install --user sphinx ghp-import==0.5.5

$ vim .zshenv
export PATH=$PATH:~/.local/bin

$ sphinx-quickstart
$ make html

change theme¶

$ pip install --user sphinx_rtd_theme

edit source/conf.py

html_theme = 'sphinx_rtd_theme'
favicon = 'favicon.ico'
pygments = 'monokai'

hosting to GitHub Pages¶

$ git init
$ git add *
$ git commit -m "first commit"
$ git remote add origin git@github.com:i0z0m/i0z0m.github.io.git
$ ghp-import -m "publish" docs/build/html
$ ghp-import -p docs/build/html
